# Ecommerce Product Card - Sony Alpha A7 Kit

Um componente de cartão de produto (Product Card) moderno, dinâmico e totalmente responsivo desenvolvido com HTML5 e CSS3. O projeto simula a exibição de um item premium de e-commerce (uma câmera fotográfica Sony), utilizando um layout dividido por formas curvas, contrastes marcantes e elementos focados em conversão.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do contêiner do produto, blocos de texto, exibição de preços e botões de ação.
* CSS3: Estilização completa, mascaramento de formas orgânicas, gerenciamento de cores e efeitos de sobreposição.
* Flexbox / CSS Grid: Organização e distribuição espacial dos blocos de informação e alinhamento dos botões.

## Funcionalidades e Técnicas Aplicadas

* Layout Assimétrico com Formas Curvas: Divisão visual interna utilizando planos de fundo curvos e orgânicos que separam a área de especificações (fundo branco) da área de destaque da marca (fundo rosa vibrante).
* Exibição de Imagem Sobreposta: Posicionamento estratégico do produto (câmera Sony) rompendo os limites das divisões internas para criar um efeito de profundidade tridimensional.
* Bloco de Informações Completo: Área dedicada ao título do produto, descrição textual detalhada, sistema de avaliação por estrelas e selo de "Free Shipping".
* Precificação de Alto Impacto: Destaque visual para o preço atualizado com desconto ($1500.00) em contraste com o preço original rasurado ($1800.00).
* Chamadas para Ação Complementares (CTA): 
  * Botão "Add To Cart" sólido e destacado em azul ciano para conversão direta.
  * Botão vazado (*ghost button*) "Login For More" na seção institucional da marca.
